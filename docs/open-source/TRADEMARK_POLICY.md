# Trademark and Brand Policy

## Brand Ownership

"Go-Fi" is the project name and brand identifier. The name is not open for unrestricted reuse.

## What Is Not Allowed

* using the Go-Fi name to imply official endorsement without permission
* reusing the logo or visual identity commercially without approval
* shipping a fork that still markets itself as the official Go-Fi product
* presenting a modified fork as if it were the original project

## Fork Rule

Forks are welcome as code forks, but they must rename the project and remove branding that could confuse users about origin or sponsorship.

## Commercial Reuse

Any commercial use of the Go-Fi name, logo, or brand assets requires permission from the project owner.

## Practical Renaming Guidance for Forks

Fork maintainers should:

* choose a new product name
* change the logo and visual identity
* update repository metadata and marketing copy
* remove references that imply official association

## Why This Matters

Trademark control protects the business even when the source code is public. It prevents clone vendors from free-riding on the reputation you build.
